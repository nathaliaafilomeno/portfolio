# Portfólio — Nathalia Filomeno

Site estático (Vite) gerado a partir do projeto no Claude Design.

## Rodar localmente

```bash
npm install
npm run dev
```

## Publicar na Vercel

1. Suba esta pasta para um repositório no GitHub.
2. Em [vercel.com](https://vercel.com), clique em **Add New → Project** e importe o repositório.
3. A Vercel detecta o Vite automaticamente (build `npm run build`, saída `dist/`). Só clicar em **Deploy**.

Alternativa sem GitHub: `npm i -g vercel && vercel` dentro desta pasta.
